모든 서브도메인 프로젝트의 Google Analytics 통계를 한 곳에서 보기 위해 만든 사이트입니다.

Google Analytics API가 Node.js를 지원하지 않기 때문에, 홈 화면을 위해 `Next.js`로 만들고 api는 `Python`으로 구성했습니다.

향후 kidow.me 도메인도 이 프로젝트로 같이 적용할 예정입니다.
